# Java基础代码学习记录

## 项目简介

这是初学Java编程语言时，跟随视频教程编写的Java基础代码练习集合。该项目涵盖了Java语言的核心基础知识，旨在帮助初学者掌握Java编程的基本概念和语法。

## 学习内容概览

### 1. 基础语法
- 数据类型和变量声明
- 运算符使用
- 控制流程（条件语句和循环语句）
- 数组操作

### 2. 面向对象编程
- 类和对象的概念与使用
- 封装、继承、多态三大特性
- 方法重载与重写
- 访问修饰符

### 3. 常用类和API
- String类及相关方法
- 集合框架基础（List、Set、Map等）
- 异常处理机制

### 4. 实用工具
- Scanner类实现控制台输入
- 基本的文件I/O操作

## 学习收获

通过这一系列的代码练习，我对Java语言有了系统的认识，掌握了编程的基本思维方法，并能够独立编写简单的Java程序。

## 后续计划

此为基础学习阶段的结束，后续将继续深入学习Java高级特性和框架技术，并将代码整理归档至GitHub以便复习和分享。
